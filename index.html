My Projects

Below are some personal projects I have been working on. Feel free to explore them on my GitHub:

Hash Validator

The Hash Validator is a tool developed in Bash to verify file integrity using hash algorithms such as MD5, BLAKE2, SHA-1, SHA-256, SHA-512224, and SHA-3. Its purpose is to ensure that files have not been altered, which is essential in cybersecurity. This tool provides a simple and efficient solution for validating file integrity on Linux and Unix-based systems.

GCS CanSat Competition

Java-based software, developed using JavaFX, was created for the CanSat 2022 competition by Virginia Tech. It visualizes real-time data from the CanSat, enables communication, simulates missions, and stores data in a database.

SQLI Lab

This repository demonstrates SQL Injection vulnerabilities in an insecure login system built with Python and Tkinter. The app connects to a MySQL database and lacks security practices like query parameterization, making it vulnerable to SQL injection. The lab illustrates the need for secure coding practices to prevent SQL injection.
